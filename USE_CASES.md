# Use Cases

1. **Pre-release gate**
   - Input: monorepo with npm + Python services.
   - Output: prioritized CVE list and release-blocking subset.

2. **Emergency patch window**
   - Input: critical RCE disclosed in transitive dependency.
   - Output: fastest safe mitigation path with rollback command set.

3. **Compliance evidence**
   - Input: quarterly security review.
   - Output: remediation status table + documented residual risks.

4. **Legacy stack cleanup**
   - Input: outdated lockfiles with many medium CVEs.
   - Output: phased upgrade plan split by risk and compatibility.
